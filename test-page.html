Great job creating your repository today on GitHub.

A repository is just a fancy word for a folder.

If you've configured GitHub Desktop, you'll be able to make a copy of the repository you created on the GitHub site and save it to your computer.

GitHub Desktop lets you save your work to the repository on your computer, and the repository on your computer is then synchronized with the corresponding repository on the GitHub website.

The process also works in reverse, i.e., you can add a file (or files) to your online GitHub repository and then use GitHub Desktop to pull those files down to the repository on your personal computer.
